# MDOptimizer
Умный скрипт для автоматической настройки, оптимизации и установки базового ПО на чистую Windows 10/11
# 🚀 MDOptimizer

![Windows](https://img.shields.io/badge/Windows-10%20%7C%2011-blue?style=for-the-badge&logo=windows)

**MDOptimizer** — это мощный и удобный Batch-скрипт для быстрой «post-install» настройки свежеустановленной Windows 10 или 11. Он экономит часы рутинной работы, автоматически скачивая нужное ПО, оптимизируя систему и настраивая обход блокировок.
