🚀 MDOptimizerMDOptimizer — это мощный и удобный Batch-скрипт для быстрой «post-install» настройки свежеустановленной Windows 10 или 11. Он экономит часы рутинной работы, автоматически скачивая нужное ПО, оптимизируя систему и настраивая обход блокировок.   __  __ ___   ___        _   _       _

✨ Что умеет скрипт?🔑 Активация Windows: Интегрированная поддержка безопасной HWID активации (через Microsoft Activation Scripts).📦 Тихая установка программ (через Winget): Steam, Discord, qBittorrent, 7-Zip.📚 Установка важных библиотек: Java 17, Visual C++ Redistributable, .NET Desktop Runtime 8, DirectX.🌍 Обход блокировок (Zapret): - Автоматическое скачивание последнего релиза Zapret с GitHub.Умная интеграция ваших пользовательских списков (lists.zip / lists.rar).Автоматическое обновление системного файла hosts для починки Telegram Web и других ресурсов.🗑️ Очистка и Debloat: Удаление телеметрии, OneDrive, Zune (Музыка/Видео), Карт, Связи с телефоном и Виджетов.⚡ Оптимизация: Включение схемы питания «Максимальная производительность» и глубокая очистка временных файлов (Temp и кэш обновлений).🛠️ Как использоватьСкачайте этот репозиторий себе на ПК (Кнопка Code -> Download ZIP).Распакуйте архив в любую удобную папку.Запустите файл MDOptimizer_Online.bat.Скрипт автоматически запросит права Администратора, необходимые для внесения изменений в систему.📁 Поддержка локальных файлов (Офлайн-режим)Твикер умеет работать с заранее скачанными файлами. Вы можете положить их в папку со скриптом, в папку Files рядом со скриптом, или на папку выше.MAS_AIO.cmd — скрипт активации (если он есть локально, твикер запустит его, а не будет качать из интернета).lists.zip или lists.rar — архив с вашими пользовательскими доменами и IP для Запрета (твикер сам распакует их в нужную папку). Для распаковки .rar требуется установленный 7-Zip.⚙️ Управление в скриптеСкрипт поддерживает два режима работы:Ручной режим (нажмите Enter при запуске): Скрипт будет останавливаться перед каждым этапом (Активация, Программы, Zapret, Очистка) и спрашивать ваше разрешение. Управление происходит клавишами Y/N (или Н/Т на русской раскладке). Нажатие Enter всегда означает «Да».Автоматический режим: Введите слово Подтверждаю на стартовом экране, и скрипт сделает всю работу полностью в фоне без лишних вопросов!⚠️ Отказ от ответственности (Disclaimer)Данный скрипт вмешивается в работу системных файлов Windows (включая файл hosts), удаляет стандартные UWP приложения и взаимодействует со скриптами обхода блокировок.
